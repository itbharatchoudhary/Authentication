-> npm create vite@latest .

1. **`npm create vite@latest .`**
   → Creates a new Vite project in the current folder (`.`) using the latest Vite version, setting up a fast React development environment.

-> npm i @react-oauth/google @tailwindcss/vite axios lucide-react react react-dom react-router-dom remixicon tailwindcss

2. **`npm i @react-oauth/google @tailwindcss/vite axios lucide-react react react-dom react-router-dom remixicon tailwindcss`**
   → Installs all the main **dependencies** your project needs:
   - `@react-oauth/google` → For Google OAuth login integration.
   - `@tailwindcss/vite` → Tailwind CSS plugin optimized for Vite.
   - `axios` → For making HTTP requests.
   - `lucide-react` → React icon library.
   - `react` → React core library.
   - `react-dom` → React DOM rendering library.
   - `react-router-dom` → For routing/navigation in React apps.
   - `remixicon` → Icon library for UI.
   - `tailwindcss` → Utility-first CSS framework.
